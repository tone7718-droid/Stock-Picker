# 08. 실전 예시로 직접 적용해보기

여기서는 가상의 간단한 회사 두 개를 만들어서 직접 계산해 봅시다.

## 예시 1: "삼성가게" (PBR이 중요한 회사)

- 연간 순이익: 200억원
- 총 주식 수: 1억 주
- 주당순이익 = 2,000원
- 주가 = 15,000원
- PER = 15,000 ÷ 2,000 = **7.5배**
- 순자산 = 3,000억원
- 주당순자산 = 3,000원
- PBR = 15,000 ÷ 3,000 = **5배**

**판단**: PER은 낮은 편이지만 PBR이 5배로 자산보다 아주 비쌉니다. 
자산보다 비싼 가게를 사는 것이에요.

## 예시 2: "성장카페" (PSR, 성장률 중요)

- 연간 매출: 500억원
- 연간 순이익: -30억원 (손실)
- 주가 = 8,000원
- 총 주식 수: 1억 주
- PSR = 8,000 / (500억 / 1억 주) = **1.6배**
- PER = 손실 중이라 계산 불가

**판단**: PER은 쓸 수 없음. 
PSR 1.6배이면 매출의 1.6배 가격. 
매출이 매년 40% 이상 증가한다면 충분히 살 수 있는 가치.

## 실전에서 하는 일

1. 한국 투자자 정보 (FnGuide, 네이버 증권, 다음 파이낸스)
2. 업종 평균 PER, PBR 검색
3. 탐색한 회사 2~3개 선정
4. 숫자 + 사업 내용 + 실적 확인
