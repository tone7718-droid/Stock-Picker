# 08. 실전 예시로 직접 적용해보기

## 예시 1: 자산집약형 제조회사

- 연간 순이익: 200억 원
- 총 주식 수: 1억 주
- 주가: 15,000원
- 자본총계: 3,000억 원

### 계산

- EPS = 200억 원 ÷ 1억 주 = **200원**
- PER = 15,000원 ÷ 200원 = **75배**
- BPS = 3,000억 원 ÷ 1억 주 = **3,000원**
- PBR = 15,000원 ÷ 3,000원 = **5배**

## 예시 2: 적자 상태의 성장회사

- 연간 매출: 500억 원
- 연간 순이익: -30억 원
- 총 주식 수: 1억 주
- 주가: 8,000원

### 계산

- 시가총액 = 8,000원 × 1억 주 = **8,000억 원**
- 주당매출 = 500억 원 ÷ 1억 주 = **500원**
- PSR = 8,000원 ÷ 500원 = **16배**

PER은 적자이므로 의미 있게 계산할 수 없습니다.

## 검산 공식

- PER = 시가총액 ÷ 연간 순이익
- PBR = 시가총액 ÷ 자본총계
- PSR = 시가총액 ÷ 연간 매출액

주당 계산과 회사 전체 계산의 결과가 같아야 합니다.

> **계산 결과보다 계산에 사용한 기간·단위·이익의 질을 확인하는 것이 먼저입니다.**
