# 주식 초보자를 위한 밸류에이션 쉽게 배우기

**PER, PBR, PSR, EV/EBITDA를 중학생도 이해할 수 있게 설명하고 실전에 적용하는 법**

구매하는 사람들이 주식 투자를 시작할 때 가장 어려운 부분이 바로 이 숫자들이에요.

"PER이 10이라는데 사는 거야?"
"PBR이 0.8이면 자산보다 싼 거 아니야?"

이 책은 그런 숫자들의 **진짜 의미**를 아주 쉽게 설명하고, 실제로 주식을 고를 때 어떻게 활용할 수 있는지 알려드립니다.

## 이 책의 특징
- 중학생도 이해할 수 있는 쉽고 단순한 설명
- 어려운 용어는 바로 풀어서 설명
- 이론만이 아니라 **실전에서 바로 적용**할 수 있게
