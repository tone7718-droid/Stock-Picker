# 주식 초보자를 위한 밸류에이션 쉽게 배우기

**PER, PBR, PSR, EV/EBITDA + 재무제표·사업 분석·위험관리까지 초보자도 이해할 수 있게 정리한 전자책**

## 이 책은

주식 투자를 이제 막 시작하는 사람을 위해 만든 **밸류에이션 배수 입문서**입니다.

어려운 용어와 공식을 최대한 줄이고, **가게 사기**, **떡볶이 가게**, **카페**, **회사 통째로 사기** 같은 일상 비유로 설명합니다.

다만 투자에서 숫자는 정답을 주는 도구가 아닙니다. 숫자는 후보를 좁히는 도구이고, 최종 판단에는 **사업의 질, 성장성, 재무 안정성, 위험관리, 심리 통제**가 함께 필요합니다.

> 이 전자책은 교육 목적의 자료입니다. 특정 종목의 매수·매도 추천이나 개인 투자 자문이 아닙니다.
