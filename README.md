# 주식 초보자를 위한 밸류에이션 쉽게 배우기

**PER, PBR, PSR, EV/EBITDA를 중학생도 이해할 수 있게 설명하고 실전에 적용하는 법**

구매하는 사람들이 주식 투자를 시작할 때 가장 어려운 부분이 바로 이 숫자들이에요.

"PER이 10이라는데 사는 거야?"
"PBR이 0.8이면 자산보다 싼 거 아니야?"

이 책은 그런 숫자들의 **진짜 의미**를 아주 쉽게 설명하고, 실제로 주식을 고를 때 어떻게 활용할 수 있는지 알려드립니다.

## 이 책의 특징
- 중학생도 이해할 수 있는 쉽고 단순한 설명
- 어려운 용어는 바로 풀어서 설명
- 이론만이 아니라 **실전에서 바로 적용**할 수 있게

## 목차

| 번호 | 제목 | 내용 |
|------|------|------|
| 01 | 서론 | 왜 밸류에이션을 알아야 할까? |
| 02 | 기본 개념 | 주식은 직장 한 조각을 사는 것 |
| 03 | PER | 이 회사는 1년에 얼마를 벌어? |
| 04 | PBR | 회사가 가진 자산을 사는 가격 |
| 05 | PSR | 아직 벌지 못하는 회사 평가하기 |
| 06 | EV/EBITDA | 회사를 통째로 사는 가격 |
| 07 | 실전 조합법 | 여러 지표를 함께 보는 법 |
| 08 | 실전 예시 | 간단한 숫자로 직접 적용해보기 |
| 09 | 함정과 주의점 | 가장 많이 하는 실수 |
| 10 | 체크리스트 | 실전에서 바로 쓸 수 있는 체크 목록 |
